# 字幕区域参数（BX、BY、BW、BH）

## 1. 作用

`BX`、`BY`、`BW`、`BH` 在 **EraseVideoSubtitles** 中共同定义一个**归一化矩形**，用于约束「仅在该区域内」进行字幕检测与擦除。矩形外的画面不应被该能力当作目标字幕区域处理（具体算法细节以厂商实现为准）。

## 2. 定义与取值范围

设视频像素宽高为 \(W \times H\)。

| 参数 | 类型 | 范围 | 几何含义 |
|------|------|------|----------|
| **BX** | Float | \([0, 1]\) | 矩形左上角 x / **视频宽度** |
| **BY** | Float | \([0, 1]\) | 矩形左上角 y / **视频高度** |
| **BW** | Float | \([0, 1]\) | 矩形宽度 / **视频宽度** |
| **BH** | Float | \([0, 1]\) | 矩形高度 / **视频高度** |

像素域左上角约为 \((BX \cdot W,\ BY \cdot H)\)，宽 \(BW \cdot W\)，高 \(BH \cdot H\)（边界与取整以服务端为准）。

## 3. 默认值（官方约定）

未显式指定时，等价于常见**底部通栏字幕带**：

| 参数 | 默认 |
|------|------|
| BX | `0` |
| BY | `0.75` |
| BW | `1` |
| BH | `0.25` |

即：自画面高度 75% 处向下取 25% 高度的全宽区域。

## 4. 与能力描述的交叉说明

- 能力说明中提到：通过抽帧在全视频范围检测擦除区域；**封面若字幕出现不足约 1 秒可能被忽略**。区域参数在此基础上进一步限制「可擦除的候选带」。
- 若实际字幕超出所设矩形，可能出现**擦除不完整**；矩形过大则可能纳入非字幕元素，需业务侧预览或后处理策略。

## 5. 实现建议

1. **UI 交互**：若提供框选，应将像素框转换为上述四个浮点并**钳制到 [0,1]**，注意视频旋转元数据导致的「显示分辨率」与「解码分辨率」不一致问题。
2. **多语言 SDK 字段名**：REST 为 `BX/BY/BW/BH`；Python 等多为 `bx/by/bw/bh`；传参前核对 SDK 模型字段。
3. **与默认行为一致**：产品若写「使用默认底部字幕区」，可不传四参数，减少与客户端浮点误差。
